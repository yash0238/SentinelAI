# WhatsApp Message Templates & Copy

> Draft copy for the Citizen Fraud Shield replies. Keep tone calm, clear, and
> non-alarmist. Never give legal or financial advice — only guidance and
> official reporting steps. Localise into the 12 supported languages.

## 1. Welcome / help

```
🛡️ SentinelAI Fraud Shield
Forward me any of these and I'll check them for you:
• A suspicious voice note
• A phone number
• A photo of a currency note
Reply in your language — Hindi, Marathi, English & more.
```

## 2. High-risk verdict (scam likely)

```
⚠️ HIGH RISK — this looks like a *{scam_type}* scam.
Do NOT: pay money, share OTP/UPI PIN, or stay on the call.
No real police or agency conducts a "digital arrest" over a video call.

Next steps:
1. Disconnect immediately.
2. Call 1930 (Cyber Crime Helpline).
3. Report at cybercrime.gov.in
Reply "REPORT" and I'll help you file it.
```

## 3. Low-risk verdict

```
✅ LOW RISK based on what you shared.
Still, stay cautious: never share OTPs or UPI PINs.
If anything feels off, call 1930.
```

## 4. Counterfeit note result

```
💵 Note check: {authenticity_verdict}
Checked: microprint {m}, security thread {t}, serial pattern {s}.
This is advisory only — confirm with your bank for anything suspicious.
```

## 5. Guided report (NCRB)

```
Let's file your report. Please share:
1. What happened (1–2 lines)
2. Any phone number / UPI id involved
3. Amount lost, if any
I'll format it for cybercrime.gov.in and give you the exact steps.
```

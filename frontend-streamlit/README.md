# SentinelAI — Streamlit Dashboard (alternative)

A Python-only dashboard you can host free on **HuggingFace Spaces**.

```bash
cd frontend-streamlit
pip install -r requirements.txt
streamlit run app.py
```

Choose **either** this or the Next.js `frontend/` — not both. See
`docs/HACKATHON_STRATEGY.md` for the tradeoff (speed vs. polish).

# Sample Data (gitignored)

Place demo inputs here. Media files are gitignored (see root `.gitignore`) —
keep them local.

```
data/
├── audio/         # genuine + synthetic voice notes (.wav/.ogg)
├── currency/      # genuine + fake note images (.jpg/.png)
└── transcripts/   # scam vs. legitimate call transcripts (.txt/.json)
```

## Rules
- **No real victim PII.** Write your own scam scripts based on public
  awareness material.
- **No redistribution of real high-value note scans.** Use RBI specimens or
  clearly-marked mock notes, kept local only.
- Label files clearly, e.g. `audio/synthetic_01.wav`, `currency/fake_500_02.jpg`.
